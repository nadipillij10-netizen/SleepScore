#Sleep Score Analysis System
Overview
Sleep Score Analysis System is a Python-based web application developed using Streamlit. The application analyzes user inputs such as age, sleep duration, stress level, and daily steps to predict sleep quality and generate a sleep score.

Technologies Used
- Python
- Streamlit
- Pandas

Features
- User-friendly web interface
- Sleep quality prediction
- Sleep score generation
- Real-time analysis

Input Parameters
- Age
- Sleep Duration
- Stress Level
- Daily Steps

Output
The system classifies sleep quality as:
- Good
- Average
- Poor

It also displays a sleep score out of 100.

How to Run
1. Install required libraries:
   pip install streamlit pandas
2. Run the application:
   streamlit run app.py
3. Open the generated local URL in a browser.

Project Objective
To provide users with a simple and interactive tool for assessing sleep quality using predefined logic based on sleep-related factors.
